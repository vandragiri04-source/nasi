# Folder Images

Simpan foto dari Instagram SMAN 33 Jakarta di folder ini.

## Cara Menggunakan:

1. Download foto dari Instagram SMAN 33 Jakarta
2. Rename file dengan format: `galeri1.jpg`, `galeri2.jpg`, dst
3. Copy ke folder ini
4. Update path di `index.html` section slideshow

## Contoh Struktur:
```
images/
├── galeri1.jpg
├── galeri2.jpg
├── galeri3.jpg
├── galeri4.jpg
└── galeri5.jpg
```

Lihat `INSTAGRAM_INTEGRATION_GUIDE.md` untuk panduan lengkap.
